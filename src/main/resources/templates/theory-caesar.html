Шифр Цезаря (Caesar Cipher)

1. Теорія шифру Цезаря

Шифр Цезаря — це один з найпростіших та найвідоміших методів симетричного шифрування, який полягає у зсуві кожної літери в алфавіті на певну кількість позицій. Названий на честь Юлія Цезаря, який використовував його для захисту військових повідомлень.

2. Переваги шифру Цезаря

  • Простота реалізації та розуміння.
  • Підходить для навчальних цілей.
  • Не вимагає складних обчислень.

3. Недоліки шифру Цезаря

  • Низька криптостійкість: лише 25 можливих ключів для латинського алфавіту.
  • Легко піддається атаці повного перебору (brute-force).
  • Непридатний для захисту реальної інформації в сучасних умовах.

4. Формула шифрування та дешифрування

  • Шифрування: C = (P + K) mod 26
  • Дешифрування: P = (C - K + 26) mod 26
  • де P — позиція літери у відкритому тексті, C — позиція літери у шифротексті, K — ключ (зсув).

5. Кроки алгоритму шифру Цезаря

  1. Обрати ключ (зсув) — ціле число від 1 до 25.
  2. Пройтися по кожному символу вхідного тексту.
  3. Якщо символ є літерою:
    • Визначити її позицію в алфавіті (0–25).
    • Додати значення зсуву (ключа).
    • Застосувати операцію mod 26, щоб залишитися в межах алфавіту.
    • Замінити оригінальну літеру на нову зашифровану.
  4. Інші символи (цифри, пробіли, пунктуація) залишаються без змін (опціонально).
  5. Об’єднати всі символи у фінальний шифротекст.

6. Реалізація шифру Цезаря на Java


public class CaesarCipher {
    public static String encrypt(String text, int shift) {
        StringBuilder result = new StringBuilder();
        shift = shift % 26;
        for (char c : text.toCharArray()) {
            if (Character.isLetter(c)) {
                char base = Character.isUpperCase(c) ? 'A' : 'a';
                char shifted = (char) ((c - base + shift) % 26 + base);
                result.append(shifted);
            } else {
                result.append(c);
            }
        }
        return result.toString();
    }

    public static String decrypt(String text, int shift) {
        return encrypt(text, 26 - shift % 26);
    }

    public static void main(String[] args) {
        String message = "HELLO";
        int key = 3;
        String encrypted = encrypt(message, key);
        String decrypted = decrypt(encrypted, key);

        System.out.println("Original: " + message);
        System.out.println("Encrypted: " + encrypted);
        System.out.println("Decrypted: " + decrypted);
    }
}
