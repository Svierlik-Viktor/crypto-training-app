SHA-256 (Secure Hash Algorithm)

1. Теорія SHA-256

SHA-256 є однією з функцій хешування, розробленою Національним інститутом стандартів і технологій (NIST). Вона належить до сімейства SHA-2 і є безпечною криптографічною функцією для захисту даних. Основна задача SHA-256 — це генерація унікального хеш-коду для кожного повідомлення.

Властивості:

  • Хеш-код має фіксовану довжину — 256 біт.
  • SHA-256 є односторонньою функцією, що ускладнює обчислення початкових даних на основі хеш-коду.
  • Захищена від колізій, що ускладнює знаходження двох різних повідомлень із однаковим хешем.

2. Переваги SHA-256

  • Безпека: Висока стійкість до колізій і атак типу "довжина розширення".
  • Простота реалізації: Підтримується стандартними бібліотеками більшості мов програмування.
  • Універсальність: Використовується у багатьох криптографічних протоколах, включаючи блокчейн.

3. Недоліки SHA-256

  • Швидкість: Повільніше, ніж деякі сучасні алгоритми хешування.
  • Обчислювальна вартість: Вимагає більше ресурсів порівняно з простими функціями хешування.

4. Приклади використання

  • Захист паролів у базах даних.
  • Перевірка цілісності файлів.
  • Використання в цифрових підписах.
  • Основний алгоритм хешування у блокчейні.

5. Реалізація SHA-256 на Java


import java.nio.charset.StandardCharsets;
import java.security.MessageDigest;
import java.security.NoSuchAlgorithmException;
import java.util.Scanner;

public class SHA256Example {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Введіть текст для хешування за допомогою SHA-256:");
        String input = scanner.nextLine();
        scanner.close();

        try {
            // Створення об'єкта MessageDigest для SHA-256
            MessageDigest digest = MessageDigest.getInstance("SHA-256");

            // Обчислення хешу
            byte[] hash = digest.digest(input.getBytes(StandardCharsets.UTF_8));

            // Перетворення байтів у шістнадцяткову строку
            String hexHash = bytesToHex(hash);

            System.out.println("Вхідний текст: " + input);
            System.out.println("Хеш (SHA-256): " + hexHash);
        } catch (NoSuchAlgorithmException e) {
            System.err.println("Алгоритм SHA-256 недоступний: " + e.getMessage());
        }
    }

    // Метод для перетворення байтового масиву у шістнадцятковий рядок
    private static String bytesToHex(byte[] bytes) {
        StringBuilder hexString = new StringBuilder();
        for (byte b : bytes) {
            String hex = Integer.toHexString(0xff & b);
            if (hex.length() == 1) hexString.append('0');
            hexString.append(hex);
        }
        return hexString.toString();
    }
}
